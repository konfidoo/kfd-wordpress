# kfd-wordpress
Wordpress integration for konfidoo


# Development
 1. `cd plugin`
 1. `npm i`
 1. `npm start`

## Start Wordpress 
 1. `docker-compose up`
 1. `open http://localhost:8080` 
 1. install wordpress
 1. go to plugins and activate

(plugin needs to be built)

# build
 1. `cd plugin`
 1. `npm i`
 1. `npm build`
